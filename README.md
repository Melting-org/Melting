# 프로젝트 소개
<img src="readmeimg/Logo.png" height="180px">
Melting은 인기 커뮤니티 사이트를 통합한 커뮤니티 사이트입니다.<br>
디시인사이드, 더쿠 등 다양한 인기 사이트를 제공하며 원하는 게시글을 스크랩 할 수 있습니다.


# 기능
- 회원가입, 로그인
- 게시글 읽기, 쓰기, 삭제, 수정, 추천, 검색, 공유
- 댓글 달기, 삭제
- 마이페이지 쓴 글 알림, 댓글 알림, 글 스크랩
